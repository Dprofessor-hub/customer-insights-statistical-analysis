#  Customer Insights – Statistical Analysis

## Project Overview

This project focuses on analyzing customer behavior using statistical techniques and data visualization. The goal is to uncover patterns in customer demographics, spending habits, and engagement, and validate insights using hypothesis testing.

---

##  Dataset Information

* **Total Records:** 10,675
* **Unique Customers:** ~1,000
* **Features Include:**

  * Demographics: Age, Gender, Education, Marital Status, State
  * Behavioral Data: Monthly Spend, Days Since Last Interaction
  * Other: Number of Pets, Transaction Dates

---

##  Tools & Technologies Used

* Python 
* Pandas
* NumPy
* Matplotlib
* Seaborn
* SciPy

---

##  Project Workflow

### 1. Data Understanding

* Checked data types, null values, duplicates
* Identified categorical and numerical variables

### 2. Descriptive Statistics

* Calculated mean, median, standard deviation
* Analyzed distribution of key variables

### 3. Data Visualization

* Histograms & Boxplots
* Bar Charts for categorical variables
* Scatter Plot (Age vs Spending)
* KDE Plot for distribution analysis

### 4. Bivariate Analysis

* Correlation matrix
* Crosstab analysis (Gender vs Marital Status)
* Grouped analysis (State, Education, Gender)

### 5. Hypothesis Testing

* **T-Test:** Gender vs Spending
* **ANOVA:** Education vs Spending
* **Chi-Square Test:** Gender vs Marital Status
* **Correlation Test:** Age vs Customer Activity

---

##  Key Insights

* Customer spending is **not significantly affected by gender or education level**
* **Marital status and gender are statistically related**
* No strong relationship between **age and customer activity**
* Spending behavior shows **high variability with presence of high-value customers**
* Customer base is **diverse and evenly distributed across demographics**

---

##  Conclusion

The analysis shows that customer behavior is largely independent of demographic factors. Businesses should focus more on behavioral segmentation and high-value customer targeting rather than relying solely on demographics.
